## 初始化顺序

```
  initLifecycle(vm)
  initEvents(vm)
  initRender(vm)
  callHook(vm, 'beforeCreate')
  initInjections(vm) // resolve injections before data/props
  initState(vm)
  initProvide(vm) // resolve provide after data/props
  callHook(vm, 'created')

  如果参数同时又render和template,会优先使用render
```

## dist结果说明
- vue.runtime.common.js(运行时  CommonJS形式)
  
- vue.common.js(运行时+编译 CommonJS形式)

- vue.runtime.esm.js(运行时+编译 ES Modules形式)

- vue.esm.js(运行时  CommonJS构建esm形式)

- vue.runtime.js(浏览器、生产环境，运行时)

- vue.runtime.min.js(浏览器、生产环境，运行时)

- vue.js(浏览器、开发环境，运行时+编译)

- vue.min.js(浏览器、生产环境，运行时+编译)
